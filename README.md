A professional, high-performance, and fully responsive multi-page website built for **Oda Special Boarding School (ODA SBS)** located in Adama, Oromia, Ethiopia.

## 📖 Background
This project was originally developed in **2022** when I was a **Grade 11 student**. I created it to bridge the gap between prospective students and the school administration, providing a digital platform for information and applications.

## ✨ Features
- **Student-Teacher Connectivity**: Dedicated pages for teachers and administrators so students can easily get in touch with their mentors and principals.
- **Online Application System**: Integrated application paths for new students to apply for admission directly through the portal.
- **Scholarship Information**: A dedicated section for the school to manage and advertise scholarship opportunities for deserving students.
- **Pre-Admission Orientation**: Comprehensive information about school rules, environment, and academic life, allowing students to know the school inside-out before they arrive.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views using Bootstrap 4.
- **Dynamic Hero Slider**: Engaging main banner with professional text animations.
- **Multi-Page Structure**: Includes specialized pages for Courses, Events, Teachers, About Us, and Notice Board.
- **Optimized Performance**: Cleaned and minified dependencies for faster load times.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript
- **Framework**: [Bootstrap 4](https://getbootstrap.com/)
- **Libraries**: 
  - [jQuery](https://jquery.com/)
  - [Slick Slider](https://kenwheeler.github.io/slick/) (Carousels)
  - [Animate.css](https://animate.style/) (CSS Animations)
  - [Themify Icons](https://themify.me/themify-icons) (Icon Font)
- **Fonts**: [Google Fonts (Poppins)](https://fonts.google.com/specimen/Poppins)

## ⚙️ Getting Started

### Prerequisites
You only need a modern web browser to view this project.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/oliftadele/ODA-SBS---Oda-Special-Boarding-School-Website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd oda-sbs-website
   ```
3. Open `index.html` in your browser.
